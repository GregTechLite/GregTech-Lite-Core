Tier Traceability Predicate

Author

Gate Guardian

This class create a new TraceabilityPredicate about IBlockTier. Used to predicates casing in MetaTileEntity via given tiers in Internal API. Here is the usage of this class:

  • Register Block Tier Hash Map via Internal API:At first, you should register a new Object2ObjectOpenHashMap in Internal API, this is the Hash Map which used to store your blocks with tier info.
  • Add terms to correspondenced Hash Map via register():Then, you should add terms like (IBlockState, IBlockTier) to your Hash Map. At this step, used enhanced for or any other workable methods are allowed too. For example, just like below: 
    
                public static final Object2ObjectOpenHashMap<IBlockState, IBlockTier>
                    MAPS = new Object2ObjectOpenHashMap();
    Content copied to clipboard
    and terms registry in register() will be: 
    
                for (BlockCasing.CasingTier tier : BlockCasing.CasingTier.values()) {
                    MAPS.put(MetaBlocks.CASING.getState(tier),
                             new WrappedIntTier(tier, tier.ordinal() + 1));
                }
    Content copied to clipboard
    where IBlockState term can be any other block class which impl Block.
  • Register Supplier via TraceabilityPredicate class:In the end, you should register a method which used to included this predicate needed in your MetaTileEntity. Pay attention, you should given a special string to declare it in MetaTileEntity data. Supplier just like: 
    
                public static Supplier<TierTraceabilityPredicate> CASING =
                    () -> new TierTraceabilityPredicate(MAPS, Comparator.comparing(
                        (s) -> ((WrappedIntTier) MAPS.get(s)).getIntTier()),
                    "Casing", null);
    Content copied to clipboard
How to used this TraceabilityPredicate in your MetaTileEntity class? At first, you should check it in formStructure() method: 

        Object type = context.get("CasingTieredStats"); // PatternMatchContext context
        this.casingTier = StructureUtility.getOrDefault(
            () -> type instanceof WrappedIntTier,
            () -> ((WrappedIntTier) type).getIntTier, 0);
        )
Content copied to clipboard
where StructureUtility is a utility class, and casingTier is a private param, and then, invalidateStructure() method used to register default value of tier, you can used like below: 

        this.casingTier = 0;
Content copied to clipboard
If you want to check tier with RecipeMap infos, then you can compare it with property of RecipeMap in checkRecipe() method: 

        return super.checkRecipe(recipe, consumeIfSuccess) // Recipe recipe, boolean consumeIfSuccess
            && recipe.getProperty(CasingTierProperty.getInstance(), 0) <= this.casingTier;

Another situation is other tier system, WrappedIntTier is example above, but this system allowed to used any tier which impl WrappedTier class.
