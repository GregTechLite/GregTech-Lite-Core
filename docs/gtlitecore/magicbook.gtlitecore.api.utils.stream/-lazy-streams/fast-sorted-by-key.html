open fun <K, V> fastSortedByKey(map: Map<K, V>, tierExtractor: ToIntFunction<in V>): List<K>

Generated a sorted list for a given map.

The action of values is from tierExtractor, it is used to send value to the sorter. As default, tier value used integer type.

Return

Sorted new list of all key values in map with the action which contained in sorter by value values in entry map of map.
