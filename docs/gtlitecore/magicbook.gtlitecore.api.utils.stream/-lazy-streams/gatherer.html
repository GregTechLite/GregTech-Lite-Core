gatherer

open fun <T, A, R> gatherer(source: Stream<T>, gatherer: Gatherer<in T, A, R>): Stream<R>

Returns a stream consisting of the results of applying the given Gatherer of the upstream source to the elements of this stream.

This is a stateful intermediate operation that is an extension point.

Gatherers are highly flexible and can describe a vast array of possibly stateful operations, with support for short-circuiting, and parallelization.

When executed in parallel, multiple intermediate results may be instantiated, populated, and merged to maintain isolation of mutable data structures. Therefore, even when executed in parallel with non-thread-safe data structures (such as a ArrayList), no additional synchronization is needed for a parallel reduction.

Return

The new stream.

source
