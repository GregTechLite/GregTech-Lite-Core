GathererSpliterator

open class GathererSpliterator<T, A, R> : Spliterator<T>

An addition spliterator for Gatherer to provide functions which the source stream unsupported with Gatherer API contents. This is a pseudo spliterator to provide a start point of stream with unsupported source contents.

This is a part of the port contents of Java 24 feature "Gatherer API" and we implement it with a spliterator and make it be the spliterator in pipeline of StreamSupport.
