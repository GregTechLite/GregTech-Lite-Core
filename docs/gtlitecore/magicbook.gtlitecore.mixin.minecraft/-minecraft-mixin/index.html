MinecraftMixin

abstract class MinecraftMixin

Feature: Open a confirmation Gui when player want to exit game to ensure they want to do it; add modpack logo to the vanilla game windows.

Like same function contents in NewHorizonsCoreMod, we shadowed the volatile param running to control the game thread on/off (in NHCore, devs used shutdown shadow and injected shutdown method, we used shadow param and overwrite shutdown method).
