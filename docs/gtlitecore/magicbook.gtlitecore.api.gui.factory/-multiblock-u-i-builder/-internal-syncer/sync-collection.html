syncCollection

open fun <T, C : Collection<T>?> syncCollection(initial: C, serializer: IByteBufSerializer<T>, deserializer: IByteBufDeserializer<T>): C

Syncs the elements of the collection to the internal buffer. On the server, elements are serialized in the order of iteration. On the client, elements are deserialized in the same order and re-added to the collection. The list will be modified on the client

Return

the synced collection

Parameters

initial

collection whose elements should be synced

serializer

serializer to write an element to the buffer

deserializer

deserializer to read the elements to add them back to the collection

<T>

collection element's type

<C>

the collection type itself
