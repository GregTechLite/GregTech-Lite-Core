Alloy Blast Smelter (ABS) is merged from Gregicality Multiblocks (GCYM), but has many different. The original idea of this RecipeMap is from same name machine in GregTech++ and Gregicality (Legacy), and ABS recipes. Different with GCYM's ABS recipes, in these recipes, fluid outputs will be fluid of material but not be molten fluid (which means player do not need to input molten fluids to Vacuum Freezer again, this is ridiculous for player experience by I think ^^). Implementation of ABS recipe automatically generator is too complex, so we just list all classes we used in this step. Pay attention, this internal API will automatically generate all materials which is allowed by AlloyBlastProducer and do not have NO_ALLOY_BLAST_RECIPES flag or DISABLE_ALLOY _PROPERTY for other addition mods.
