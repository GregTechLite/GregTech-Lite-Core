ItemStackMap

class ItemStackMap<T> : AbstractMap<K, V>

A map class for ItemStack keys with wildcard damage/NBT. Optimised for lookup.

Author

CodeChiken, glee8e, mitchej123

This map does NOT support null values or null keys! null will be silently ignored! Originally created by CodeChicken for NotEnoughItems. Adapted to Map<ItemStack, T> interface by glee8e. Edited later to add the option to ignore NBT tags to match vanilla furnace recipe map behavior.

Since

2.8.7-beta
